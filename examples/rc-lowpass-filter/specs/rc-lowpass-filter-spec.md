## 要求仕様

### 機能要件
- [x] 1次 RC ローパスフィルタであること
- [x] `VIN` / `VOUT` / `GND` を named net として明示できること
- [x] 外部入出力をコネクタとしてモデル化すること
- [x] カットオフ周波数 1kHz を標準値部品で近似すること

### 電気的仕様
| 項目 | 最小 | 標準 | 最大 | 単位 |
|------|------|------|------|------|
| 入力信号振幅 | 0.1 | 1.0 | 3.3 | V |
| 入力源インピーダンス | - | 50 | 100 | ohm |
| カットオフ周波数 | 950 | 995 | 1050 | Hz |
| 通過帯域利得 | -0.5 | 0.0 | 0.0 | dB |
| 動作温度 | 0 | 25 | 50 | degC |

### 部品選定方針
- **抵抗**: 0603 サイズ、1%、1/10W
- **コンデンサ**: 0603 または 0805 の積層セラミック、X7R、50V 定格
- **コネクタ**: 2.54mm ピンヘッダ 1x1 を入力、出力、GND テストポイント用途に使う

### インターフェース
- **入力**: `VIN` / `GND`
- **出力**: `VOUT` / `GND`
- **基準電位**: `GND`

## 設計制約
- [x] カットオフ周波数は `fc = 1 / (2piRC)` で算出すること
- [x] コンデンサ値は 100nF を固定し、抵抗は E24 系列から選定すること
- [x] 受動部品のみで構成し、増幅素子は含めないこと
- [x] 外部 I/O はコネクタまたはテストポイントとして部品化すること

## 設計計算

### RC 定数
```
fc = 1 / (2piRC)
R = 1 / (2pi x 1000Hz x 100nF)
R = 1591.5ohm

E24 系列選定: 1.6kΩ
実際の fc = 1 / (2pi x 1.6kΩ x 100nF) = 994.7Hz
```
