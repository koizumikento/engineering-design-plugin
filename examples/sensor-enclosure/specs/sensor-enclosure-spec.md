| 項目 | 値 | 公差 | 備考 |
|------|-----|------|------|
| 全長 | 60mm | ±0.5mm | 外形寸法 |
| 全幅 | 40mm | ±0.5mm | 外形寸法 |
| 全高 | 25mm | ±0.5mm | 蓋含む |
| 肉厚 | 2.0mm | ±0.2mm | 均一 |

| 名称 | 種類 | 寸法 | 位置 | 備考 |

| ケーブルグランド | 貫通穴 | φ12.5mm | 短辺中央 | PG7用 |

- **壁掛け**: 長穴×2（ねじ頭5mm用）
