# Calibration Block Specification

## 目的

CadQuery runner のSTEP-first出力、検査レポート、複数視点プレビューを確認するための小型校正ブロック。

## 寸法

- 外形: 40.0 x 30.0 x 12.0 mm
- 貫通穴: M3クリアランス相当、直径 3.4 mm
- 穴位置: 中心から X方向 +/- 12.0 mm、Y方向 0.0 mm
- 上面ポケット: 20.0 x 10.0 mm、深さ 2.0 mm
- 外周フィレット: 1.0 mm
- 上面ポケット角: 直角

## 出力

- [x] STEP
- [x] STL
- [x] JSON検査レポート
- [x] PNG複数視点プレビュー

## 承認

- [x] **仕様承認済み** - 承認後に `mechanical-cad` スキルでコード生成を実行
