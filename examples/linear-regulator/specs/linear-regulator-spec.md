# 回路設計仕様書 - 線形レギュレータ

## 基本情報
- **プロジェクト名**: linear-regulator
- **バージョン**: 1.0
- **作成日**: 2026-03-07
- **ステータス**: Approved

## 要求仕様

### 機能要件
- [x] 9V〜15V の DC 入力から 5V を生成する
- [x] 一般的な 3 端子レギュレータを使う
- [x] 入力側と出力側に安定化用コンデンサを入れる

### 電気的仕様
| 項目 | 最小 | 標準 | 最大 | 単位 |
|------|------|------|------|------|
| 入力電圧 | 9.0 | 12.0 | 15.0 | V |
| 出力電圧 | 4.8 | 5.0 | 5.2 | V |
| 出力電流 | - | 100 | 250 | mA |

### 部品選定方針
- **レギュレータ**: `L7805`
- **入力コンデンサ**: 電解 10uF + セラミック 100nF
- **出力コンデンサ**: 電解 10uF + セラミック 100nF
- **コネクタ**: 入出力とも 2 ピンヘッダ

### インターフェース
- **入力**: `VIN`, `GND`
- **出力**: `VOUT`, `GND`

## 設計制約
- [x] KiCad 標準ライブラリの部品で構成する
- [x] 外部 I/O はコネクタでモデル化する
- [x] SKiDL の ERC を通す

## 設計メモ
- 5V 出力の簡易サンプルとして、扱いやすい `L7805` を採用する
- 熱設計や放熱器選定はこのサンプルの範囲外とする
- KiCad native exporter は現状このトポロジーに未対応のため、SKiDL/E RC/BOM 向け sample として扱う

## シミュレーション要件
- [ ] 必須ではない
- [x] まずは ERC と部品表生成を優先する

## 出力形式
- [x] Python スクリプト
- [x] BOM / ERC summary / 設計メモの元になる SKiDL 回路
- [ ] KiCad v9 ネイティブ回路図は exporter 拡張後に対応
