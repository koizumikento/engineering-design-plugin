# 回路設計仕様書

## 基本情報
- **プロジェクト名**: Inverting Amplifier
- **バージョン**: 1.0
- **作成日**: 2026-03-07
- **ステータス**: Approved

## 要求仕様

### 機能要件
- [x] TL072 を使用した反転増幅回路であること
- [x] 電圧ゲインが -10 V/V であること
- [x] 入出力と電源を named net として明示できること
- [x] 未使用のオペアンプチャネルを安全に終端すること
- [x] 電源デカップリングコンデンサを含むこと

### 電気的仕様
| 項目 | 最小 | 標準 | 最大 | 単位 |
|------|------|------|------|------|
| 正電源電圧 | 10 | 12 | 15 | V |
| 負電源電圧 | -15 | -12 | -10 | V |
| 入力信号振幅 | -0.5 | 0.1 | 0.5 | V |
| 電圧ゲイン | -10.1 | -10.0 | -9.9 | V/V |
| 出力信号振幅 | -5.0 | -1.0 | 5.0 | V |
| 動作温度 | 0 | 25 | 50 | °C |

### 部品選定方針
- **抵抗**: 0603 サイズ、1%、1/10W
- **コンデンサ**: 0603 または 0805 の積層セラミック、X7R、50V 定格
- **半導体**: 汎用 JFET 入力オペアンプ TL072、DIP-8 パッケージ

### インターフェース
- **入力**: `VIN` / `GND` の named net
- **出力**: `VOUT` / `GND` の named net
- **電源**: `VCC` / `GND` / `VEE` の named net

## 設計制約
- [x] 入力抵抗と帰還抵抗は標準値の組み合わせでゲイン -10 を実現すること
- [x] 非反転入力は `GND` 基準で固定すること
- [x] 2 回路入りオペアンプの未使用チャネルを浮かせないこと
- [x] 正負電源それぞれに 100nF のローカルデカップリングを配置すること

## 備考
- 既存の非反転増幅サンプルと比較しやすいように、同じ TL072 と同じ電源構成を採用する。
- 反転入力ノードは外部ラベルだけで済ませず、入力コネクタと出力コネクタを部品として配置する。
