## 要求仕様

### 機能要件
- [x] 1段あたり 4 台、全 5 段で合計 20 台のバッテリを搭載できること
- [x] 各段は 2 x 2 配置として、左右方向と前後方向の重心偏りを抑えること
- [x] 最下段を低くし、上段ほど背面側へセットバックさせて、満載時の重心を支持範囲内へ収めること
- [x] ベースの前後張り出し、側板、横桟により静的に安定した自立ラック形状とすること

## 設計制約
- [x] 満載時でもバッテリ重心投影がベース支持範囲内に入ること
- [x] 上段ほど背面へ 8 mm ずつセットバックさせること
- [x] 前面は低い落下防止リップとし、電池の出し入れを阻害しないこと
- [x] CAD モデルは組立表現ではなく単一ソリッドとして STEP/STL 出力できること

## 出力形式
- [x] STEP
- [x] STL
- [x] PNG（プレビュー）

## 備考
- 本仕様はユーザー寸法未提示のため、試作用として一般的な 12V 7Ah バッテリ寸法を仮定して作成した。
- 重心評価はバッテリ 20 台を搭載した静的状態を対象とし、動的荷重や地震荷重は含めない。
