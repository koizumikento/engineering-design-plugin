- **プロジェクト名**: voltage-divider
- **バージョン**: 1.0
- **作成日**: 2026-03-07
- **ステータス**: Approved

## 要求仕様

### 機能要件
- [x] 5V 入力から 3.3V 系の分圧出力を得る抵抗分圧回路であること
- [x] `VIN` / `VOUT` / `GND` を named net として明示すること
- [x] 外部入出力をコネクタでモデル化すること
- [x] E24 系列の標準抵抗値を使用すること

### 電気的仕様
| 項目 | 最小 | 標準 | 最大 | 単位 |
|------|------|------|------|------|
| 入力電圧 | 4.75 | 5.0 | 5.25 | V |
| 出力電圧 | 3.20 | 3.33 | 3.47 | V |
| 分圧電流 | 1.50 | 1.67 | 1.85 | mA |
| 抵抗 R1 (`VIN` 側) | 1.0 | 1.0 | 1.0 | kOhm |
| 抵抗 R2 (`GND` 側) | 2.0 | 2.0 | 2.0 | kOhm |

### 部品選定方針
- **抵抗**: 0603 サイズ、1%、1/10W
- **コネクタ**: 2.54mm 単列ピンヘッダ相当

### インターフェース
- **入力**: `VIN`
- **出力**: `VOUT`
- **基準電位**: `GND`

## 設計制約
- [x] 抵抗比 1:2 で 5V から 3.33V を得ること
- [x] 回路は 2 抵抗のみの最小構成とすること
- [x] 後段負荷は高インピーダンス入力を前提とすること
