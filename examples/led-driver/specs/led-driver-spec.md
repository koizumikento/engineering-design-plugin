# 回路設計仕様書 - LEDドライバ

## 基本情報
- **プロジェクト名**: led-driver
- **バージョン**: 1.0

- **ステータス**: Approved

## 要求仕様

### 機能要件
- [x] 白色LED 3個を並列駆動
- [x] 各LEDに個別の電流制限
- [x] 5V単一電源動作

### 電気的仕様
| 項目 | 最小 | 標準 | 最大 | 単位 |
|------|------|------|------|------|
| 入力電圧 | 4.5 | 5.0 | 5.5 | V |
| LED順方向電圧 | 3.0 | 3.2 | 3.4 | V |
| LED順方向電流 | - | 20 | 25 | mA |
| 総消費電流 | - | 60 | 75 | mA |

### 部品選定方針
- **抵抗**: 0603サイズ、1%、1/10W
- **LED**: 白色、5mm砲弾型または0805SMD
- **コネクタ**: 2ピンヘッダ（電源用）

### インターフェース
- **電源入力**: +5V / GND（2ピンヘッダ）
- **出力**: LED 3個（基板実装）

## 設計制約
- [x] 個別電流制限（LED故障時に他に影響なし）
- [x] 一般的な部品のみ使用

## 設計計算

### 電流制限抵抗
```
R = (Vin - Vf) / If
R = (5.0 - 3.2) / 0.020
R = 90Ω

E24系列選定: 91Ω または 100Ω
100Ωを選定: If = 1.8V / 100Ω = 18mA
```
