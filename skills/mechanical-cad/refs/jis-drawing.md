| 用途 | サイズ |
|------|--------|
| 外観エッジ | R1〜R3 |
| 構造コーナー | R = 肉厚の0.5倍以上 |

```

```

## 7. 防水等級（JIS C 0920 / IEC 60529）

| 等級 | 名称 | 試験条件 |
|------|------|---------|
| IP54 | 防塵・防滴 | 粉塵侵入を防止、あらゆる方向からの水滴 |
| IP65 | 防塵・防噴流 | 粉塵完全保護、あらゆる方向からの噴流水 |
| IP67 | 防塵・防浸 | 粉塵完全保護、水深1m・30分 |
| IP68 | 防塵・水中 | 粉塵完全保護、指定条件での水中使用 |

### 防水設計のポイント

- Oリング溝: JIS B 2401準拠
- パッキン面: Ra1.6以下
- ねじ部: シール剤またはOリング
